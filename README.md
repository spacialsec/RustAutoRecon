# RustAutoRecon
A blazing fast implementation of AutoRecon in Rust.  A multi-threaded network reconnaissance tool which performs automated enumeration of services.  Works up to 20% compared to the python implementation.

# Usage
Compile the tool
```bash
rustc autorecon.rs
```

Run the tool
```bash
./autorecon --flags ...
```
